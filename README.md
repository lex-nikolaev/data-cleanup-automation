# Data Cleanup Automation Pipeline
Automated ingestion invoices CSVs into a structured SQL database using Python/Pandas

### 🛠 Solution
I developed a Python-based automation tool that:
* **Extracts:** Scans a directory for multiple CSV/Excel sources.
* **Transforms:** Cleans headers, standardizes date formats, and handles missing values using Pandas.
* **Loads:** Consolidates data into a single, analysis-ready Master Table (Excel/SQL).

### 📈 Results
* Replaced 10+ hours of manual weekly work with a 30-second script.
* Eliminated human error in data consolidation.
* Enabled real-time sales tracking via Power BI.

### 💻 Tech Stack
* Python (Pandas, NumPy) 
* Logging & Error Handling
